
# README

Rails Todo List
Written on Ruby 2.6.7 & Rails 6.1

## Installation

0. Install Ruby & Rails if not done so already
1. Clone Repo
2. Run `bundle install`
3. Run `rake db:migrate`
4. Open `localhost`
5. Start making your Todo list
